# Security Policy

## Supported Versions

The table below lists which versions of this project currently receive security
updates. At this time, only the 5.1.x and 4.0.x release lines are supported.

| Version        | Supported          |
| -------------- | ------------------ |
| Latest release | :white_check_mark: |
| Earlier versions | :x:              |
